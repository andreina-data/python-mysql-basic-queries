Este proyecto es una práctica de integración entre Python y MySQL, donde se establece una conexión a una base de datos, se ejecutan consultas SQL y se realizan inserciones de registros desde Python.

El objetivo de este proyecto fue aplicar operaciones básicas y el manejo de datos desde scripts en Python.

Se usó Python 3.x, MySQL y la librería mysql-connector-python

Serie de proyectos:
Este proyecto pertenece a una serie de prácticas enfocadas en la integración de Python con diferentes tecnologías para el acceso, almacenamiento y procesamiento de datos. Cada repositorio aborda un objetivo específico y complementa el aprendizaje de la serie.

- Python + MySQL | Consultas basicas
- Python + MySQL | Creación y carga de bases de datos
- Python + API + MongoDB
